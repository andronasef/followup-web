| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-01-08 | Tampering | src/server/repo/*.ts | high | mitigate | Every query uses Drizzle's parameterized query builder or postgres.js tagged-template `sql` calls — never string interpolation of caller-supplied values into a query string |
| T-01-09 | Denial of Service | src/server/repo/ratelimit.ts | medium | mitigate | Race-free single-statement token bucket (Pattern 7) prevents concurrent-request bypass of the rate limit under load |
